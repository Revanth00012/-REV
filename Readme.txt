Final Project running insturctions:

Team name (Ignited minds)
Instructions:

1.Install Node.js and set up the environment
2.Install Tomcat Server and set it up for https
3.Install Mongo DB Database and set up the database
4.Keep the IgnitedMinds Folder in the webapps of Tomcat server.
5.Restart the Tomcat server
6.Go to the WPL Project Folder from command line and type "node server.js" to start the app server
7.Open chrome and go to https://localhost:8443/IgnitedMinds
8.Thus you can see the demonstration on the for the final project.
